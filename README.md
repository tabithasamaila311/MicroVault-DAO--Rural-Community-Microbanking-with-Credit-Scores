# MicroVault DAO: Rural Community Microbanking with Credit Scores
MicroVault DAO is a decentralized microbanking platform enabling rural communities to access financial services through a credit score-based lending system.

## ✨ Core Features
- 💳 Credit Score System (500-1000 points)
- 💰 Decentralized Lending
- 📊 Loan Management
- 🔐 Secure User Profiles

## 🛠 Technical Implementation
- Smart Contract: Clarity
- Testing Framework: Clarinet
- Blockchain: Stacks

## 📋 Contract Functions
1. **User Management**
   - register-user
   - get-user-data

2. **Loan Operations**
   - request-loan
   - repay-loan
   - get-loan-data

3. **Admin Controls**
   - update-min-credit-score

## 🚀 Getting Started

### Prerequisites
```bash
clarinet new microvault-dao
```

### Deploy Contract
```bash
clarinet deploy
```

### Run Tests
```bash
clarinet test
```
